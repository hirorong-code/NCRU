# Phase Mapping Network Theory (PMNT) Rev.1.12
## 宇宙OS：最小計算資源単位（NCRU）による時空の動態的構成

### 1. 背景と動機
現代物理学における「時空」を、背景形式（器）としてではなく、離散的な計算資源の配分結果として再定義する。本論文は、宇宙を巨大な並列演算処理系と見なし、記述コストの最小化原理から物理法則が創発するプロセスを記述する。

### 2. 基本原理
- **NCRU (Narrowest Computational Resource Unit)**: 
  宇宙を構成する最小の演算単位。各ノードは有限の処理容量（スループット）を持つ。
- **最小計算負荷原理 ($\delta I = 0$)**: 
  系の遷移は、情報の記述コスト $I$ が最小となる経路を選択する。これがフェルマーの原理や最小作用の原理の基底となる。
- **エネルギーの再定義**: 
  ノードあたりの「書き換え密度」。
- **重力と時間**: 
  局所的な計算負荷の増大（渋滞）が、クロックサイクルの遅延を引き起こす。これが「時間の遅れ」および「空間の歪み」として観測される。

### 3. 数理モデル
情報パルスがNCRUネットワークを通過する際の有効クロック $\Delta \tau_{eff}$ は、局所負荷 $\Phi$ に対して以下の指数関数的減衰を示す：

$$\Delta \tau_{eff} \propto \exp(-2\Phi)$$

これにより、高負荷領域（重力源）付近での記述スループットが低下し、観測的には「空間の曲率」として解釈される。

### 4. 実装：演算動態写像
本リポジトリに同梱の `index.html` は、上記原理を $O(n^2)$ の演算系でシミュレートしたものである。
- **白パルス**: 自由伝播する情報（光子・粒子）。
- **赤色ノード**: 高負荷（高密度記述）領域。
- **挙動**: パルスが赤色領域を通過する際、$\Delta \tau_{eff}$ の低下により速度が減衰し、最短経路（記述コスト最小化）を求めるように屈曲する。
