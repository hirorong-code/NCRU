## Abstract

本論文は、ダイヤモンドNVセンターの大離調Tavis-Cummingsモデルに基づく量子ネットワーク（PMNT GEN3）が解ける問題と解けない問題の境界を初めて定義する。

GEN3は手順を実行しない。J_{ij}重み行列をコスト関数として与えると、物理系がエネルギー最小化を通じてネットワーク構造を自発的に形成し、答えを出力する。「手順ではなく物理が解く」——これがPMNTの計算原理である。

L31〜L35の数値シミュレーション（量子Lindbladマスター方程式）を通じて以下が確定した：

**計算できること：**  
J_{ij}行列のクラスター固有構造の抽出。タンパク質複合体の自動検出（Scenario1：2複合体、Scenario3：隠れた三量体）で実証。NANDなし、クロックなし、ラベルなし。

**計算できないこと：**  
縮退したJ_{ij}を持つペアの区別（Scenario2）、符号反転が必要な問題（MAX-CUT）、中間値の完全順位復元（L34）。

この境界は「失敗」ではなく「計算クラスの定義」である。GEN3は「J_{ij}行列のスペクトルクラスター構造を物理的に抽出する計算機」として完全に定義された。

### 1.1 ノイマン型との本質的な違い

```
ノイマン型計算機：

  機械がその通りに実行する
  「計算とは手順の実行である」

  人間がJ_{ij}（コスト関数）を設定する

  答えがネットワーク構造として現れる
  「計算とは物理の自然な帰結である」
```

この違いは本質的である。ノイマン型では「何をするか」を人間が記述する。PMNTでは「どうあるか」を人間が設定し、「どうなるか」を物理が決める。

```

        + H_Γ（横磁場項）

J_eff = g²/Δ = 308 MHz
κ_eff = κ(g/Δ)² = 8.85 MHz
```

**確定パラメータ（L33）：**

## 2. 計算できること

### 2.1 クラスター構造の検出（Scenario1）

**問題設定：**  
タンパク質A〜Dの相互作用強度J_{ij}から複合体構造を自動検出。

```
入力J_{ij}：
  J_AB = 1.0（複合体I内）
  J_CD = 1.0（複合体II内）
  他   = 0.1（複合体間）

正解：複合体I = {A,B}、複合体II = {C,D}
```

**GEN3の出力（s*=0.890で読み出し）：**

```
C_AB = C_CD = +0.2347（複合体内）
その他      = +0.1929（複合体間）
分離マージン = +0.0418
```

**判定：✓ DETECTED**  
物理は手順なしで2つの複合体を正確に発見した。ラベルを与えていない。J_{ij}の構造だけが入力である。

**生物学的意義：**  
ヘモグロビン（αβ二量体×2）的な構造の自動同定。質量分析や共免疫沈降なしに、相互作用強度データだけから複合体構造が得られる。

### 2.2 隠れた構造の発見（Scenario3）

**問題設定：**  
J_{ij}の配置が一見非自明な系で、{A,B,D}三量体複合体を検出。

```
入力J_{ij}：
  J_AB = 0.8、J_AD = 0.7、J_BD = 0.6
  J_AC = J_BC = J_CD = 0.1（孤立ProtC）

「隠れた」三量体：{A,B,D}
```

**GEN3の出力：**

```
C_AB = +0.237、C_AD = +0.204、C_BD = +0.184
C_CD = +0.112、C_BC = +0.093、C_AC = +0.088

複合体内平均C = +0.208
複合体間平均C = +0.097
分離マージン  = +0.072（最大）
```

**判定：✓ DETECTED**  
「隠れた」と言ったのは人間だけである。物理には最初から構造が見えていた。

**重要な発見：**  
Scenario1（マージン+0.042）よりScenario3（マージン+0.072）の方がマージンが大きい。これはJ_{ij}の非対称性が大きいほど識別が容易であることを示す。構造が「隠れている」ことは問題の難しさと無関係である。

## 3. 計算できないこと

### 3.1 縮退問題（Scenario2）

**問題設定：**

```
J_AB = 1.0（強）
J_BC = 0.5（橋渡し）
J_AC = 0.5（橋渡し）← J_BCと同値
J_CD = 0.8（中強）
J_AD = J_BD = 0.1（弱）
```

**GEN3の出力：**

```
C_AC = C_BC = +0.1602（完全に同じ値）
```

**物理的理由：**

```
J_AC = J_BC
→ H_effにおけるA-CペアとB-Cペアの
  エネルギーが同じ
→ C_{ij}も同じ値に収束
= 縮退

物理は「どちらが複合体内か」を知らない
知る必要がない
J_{ij}の値が同じなら区別する理由がない
```

これは制限ではなく物理の誠実な答えである。縮退を解除するには問題の設定（J_{ij}）を変えるか、系のサイズN_SPINを増やして文脈情報を加える必要がある。

### 3.2 符号問題

現行GEN3のH_XYは強磁性型：

```
H = -J_eff(σ_+σ_- + σ_-σ_+)
→ 全C_{ij} > 0（常に正値）
```

符号反転（C_{ij} < 0）が必要な問題、例えばMAX-CUTは解けない。2層PMNTまたは反強磁性型H_XYの実装が必要。

### 3.3 順序の完全復元（L34）

J_{ij}が6段階の異なる値を持つとき（L34 Pattern1）、C_{ij}の順位がJ_{ij}の順位と一致しない（Kendall tau = -0.600）。

**物理的理由：**

```
C_{ij} ∝ sin²(J_eff × J_{ij} × t*)

sin²は非単調関数
J_{ij}が大きすぎると
sin²が折り返してC_{ij}が減少

= 動作範囲の制約：
  J_eff × J_{ij} × t* < π/4
  を満たすJ_{ij}の範囲内でのみ
  単調な対応が成立
```

```

  出力：C_{ij}（相関行列）

  解ける：
  J_{ij}行列のクラスター固有構造が

  正確に反映される問題

  条件：
  ① 異なるクラスター間でJ_{ij}の値が
    明確に分離している（非縮退）
  ② 動作範囲：J_eff × J_{ij} × t* < π/4
  ③ 符号反転が不要
```

```
GEN3は「スペクトルクラスタリングの
物理的アナログ」
```

この対応は重要である。GEN3が解いている問題が古典的に定義された計算問題と一致することを意味し、「物理が計算している」という主張に具体的な内容を与える。
