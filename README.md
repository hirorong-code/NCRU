### 【理論のコア / Core Principles】
- **重力 (Gravity)**: 計算資源の局所的渋滞によるスループット（クロック）の低下。
- **エネルギー (Energy)**: NCRUノードが1クロックあたりに処理可能な最大書き換え容量。
- **最小計算負荷原理 (Minimal Computational Load)**: 宇宙の状態遷移は記述コスト $I$ を最小化する経路を選択する ($\delta I = 0$)。
