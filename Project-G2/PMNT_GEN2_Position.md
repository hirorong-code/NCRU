## 3. 文献で確定した理論的根拠

### 3-1. 熱支援量子アニーリング（QTA）の実験的実証

**Dickson et al., Nature Communications 4, 1903 (2013)**

16量子ビット超伝導プロセッサで、熱環境が量子アニーリングを
最大1000倍高速化することを実証。

機構：
```
アンティクロッシング（小さなエネルギーギャップ）で：
  熱励起   → 障壁手前で系を励起（前半）
  量子トンネリング → 障壁を抜ける
  熱緩和   → 基底状態に落ちる（後半）

デコヒーレンス時間より8桁長いアニーリング時間でも
完全コヒーレント系と同等の成功確率を達成
→ 完全な量子コヒーレンスは不要
→ 熱浴に接続されたまま計算できる
```

最適温度 T_peak の存在：
```
T_peak = δE / k_B

δE = 問題ハミルトニアンのエネルギーギャップ
→ 問題ごとに「ちょうど良い温度」が存在する
→ 固定温度（D-Wave: 15mK）では最適化できない問題がある
→ 可変温度（PMNT: 20K〜80K）が理論的に有利
```

### 3-2. D-Waveの根本的限界（PMNTとの差別化根拠）

**Albash et al., Phys. Rev. Lett. 119, 110502 (2017)**

温度スケーリング則：
```
固定温度の量子アニーラーには根本的な限界がある。
最適化器として機能するには、問題サイズNに対して：

  T ∝ 1/log(N)  （最低限）
  T ∝ 1/N^β     （最悪ケース）

で温度を下げる必要がある。

D-Waveの問題：
  15mK で固定 → 問題が大きくなるほど性能劣化
  これ以上冷やすのは物理的に困難
  → スケーリングの壁に当たっている

PMNT GEN2の回答：
  動作温度を問題に応じて 20K〜80K で可変制御
  温度スケーリング則を動的に満たす
  液体窒素（77K）は扱いが容易で現実的
```

### 3-3. QTAが古典SAに優る問題クラス（最重要）

**Baldassi & Zecchina, PNAS 115, 1457 (2018)**

```
機械学習に中心的な問題クラスで QTA が SA に指数的優位。

SA（古典熱アニーリング）が苦手：
  「高くて薄い障壁に囲まれた浅い局所最小」
  熱励起確率 ∝ exp(-ΔE/kT) → 障壁が高いと指数的に困難

QTA（量子熱アニーリング）が得意：
  量子トンネリング確率は障壁の「幅」に依存
  高くても薄い障壁なら抜けられる
  機械学習の損失景観はこの構造を持つ

→ PMNTが最も相性の良い問題クラス：
  機械学習的最適化問題
  （MAX-CUT、スピングラス、組み合わせ最適化）
```

### 3-4. NVセンターのT₂温度依存性（実験値確定）

**Bar-Gill et al., Nature Communications 4, 1743 (2013)**

| 温度 | T₂ | 律速要因 | 備考 |
|------|-----|---------|------|
| 300K | 184μs〜2.4ms | フォノン＋P1 | GEN1の限界 |
| 77K | **≈0.6秒** | フォノン→¹³C | **300K比×300倍** |
| 5K | 0.74ms | ¹³C核スピン拡散 | 同位体精製で改善 |

```
重要な発見：
  77K で T₂ ≈ 0.6s（×300倍改善）
  5K 以下では ¹³C が律速 → 冷やしすぎても改善しない
  同位体精製（¹²C）なら 10K〜80K で T₂ ≫ 1s の可能性

最適動作温度域：20K〜80K
  熱揺らぎ：kT = 1.7〜6.6 meV（J と同程度）
  量子コヒーレンス：T₂ = 0.6s 以上
  液体窒素（77K）で達成可能
```

## 5. GEN2の計算原理（定式化）

```
ハミルトニアン：
  H = -Σᵢⱼ Wᵢⱼ σᵢ σⱼ - Γ(t) Σᵢ σᵢˣ

  Wᵢⱼ   ：NV-空孔配置で決まる結合（製造時固定）
  σᵢ=±1 ：NVスピン状態（ms=±1）
  Γ(t)  ：横磁場＝量子揺らぎの強さ（外部磁場で制御）

アニーリングスケジュール：
  初期：Γ 大 → 量子揺らぎ支配・全状態の重ね合わせ
  中間：kT ~ J ← PMNTの主動作域（熱と量子の協調）
  最終：Γ → 0 → 古典イジング・解が確定

最適動作条件：
  T = T_peak = δE / k_B（問題のエネルギーギャップに依存）
  τ_anneal < T₂(77K) = 0.6s ← 量子コヒーレンスが保持

読み出し：
  ODMR → 全NVのms状態 → イジングスピン配置 → 最適解
```

## 7. D-Waveとの本質的差異（一言）

```
D-Wave  ：「熱を排除して量子だけで解く」
           → 問題が大きくなるほど冷やす必要
           → 既に物理的限界（15mK）に近い

PMNT GEN2：「熱を使いながら量子で局所最小を脱出する」
            → 問題に応じて最適温度を選ぶ
            → 液体窒素（77K）で十分

これは原理の違い：
  「環境（熱）を敵とする」 vs 「環境（熱）を味方にする」

PMNTが当初から持っていた発想（熱エネルギーが駆動力）が
量子アニーリングの最先端理論（Dickson 2013）と一致していた。
```
