# エネルギー最小化としての計算の最小スカラー定式化

## 幾何を排除したPMNTの核心モデル

## 要旨

本稿は、幾何学・計量・ヒルベルト空間に依存しないPhase Mapping Network
Theory (PMNT)
の最小閉形式モデルを提示する。計算は勾配流に従う自律的エネルギー最小化として定義される。制御依存二次係数を持つ四次スカラー汎関数のみで、状態選択および構造転移が実装可能であることを示す。

------------------------------------------------------------------------

## 1. 動機

従来の物理学は幾何構造を一次とする。\
PMNTは次を公理とする。

**公理：宇宙はスカラー整合度関数を最小化する方向に進化する。**

------------------------------------------------------------------------

## 2. 公理系

### 公理1 --- スカラー状態変数

系は実数スカラー φ ∈ ℝ で記述される。

### 公理2 --- 整合度関数

H(φ;λ) ≡ 𝒞(φ;λ)

### 公理3 --- 勾配ダイナミクス

dφ/dt = -∂H/∂φ

------------------------------------------------------------------------

## 3. 最小閉形式

H(φ;λ) = a(λ) φ² + b φ⁴\
b \> 0

------------------------------------------------------------------------

## 4. 状態選択＝計算

定常条件: ∂H/∂φ = 2aφ + 4bφ³ = 0

解: - φ = 0 - φ = ±√(-a/2b) (a \< 0 の場合)

緩和により極小が自律的に選択される。\
これを計算と定義する。

------------------------------------------------------------------------

## 5. 構造転移

臨界条件: a(λ_c) = 0

この点でエネルギー地形のトポロジーが変化する。

------------------------------------------------------------------------

## 6. 幾何の不在

本モデルは以下を含まない: - 計量テンソル - 多様体 - ヒルベルト空間 -
座標依存性

計算は純粋にスカラー地形から生じる。

------------------------------------------------------------------------

## 7. 中心定理

四次スカラー汎関数と勾配ダイナミクスのみで、幾何構造を仮定せずに計算を実装できる。

------------------------------------------------------------------------

## 8. 結論

計算 ≡ エネルギー最小化
