# PMNT-OQC 要件定義書 (Requirements Specification) Rev.1.95
**プロジェクト:** Phase Mapping Network Theory - Optimized Quantum Computation
**ステータス:** 最終確定版 (Final Canon)

## 0. 公理的基盤 (Axiomatic Foundation)
1. **J-最小化原理:** 物理的緩和（Lyapunov性）を演算の動力源とする。
2. **Fisher情報量の消失:** 情報生成の停止（I(θ) → 0）を演算の完結と定義する。
3. **4.25 eV 閉包条件:** 外界からの因果・エントロピー流入を遮断するエネルギー障壁。
4. **観測的凍結:** 収束状態を破壊せず、エネルギー勾配に直交する手法で位相をラッチする。
5. **意味論的相似性:** 論理空間の最適解と、物理的なエネルギーの「谷」を相似形に配線する。

## 1. 材質仕様 (Material Specifications)
### 1.1 基底材質: Type IIa 単結晶ダイヤモンド
* **選定理由:** バンドギャップ 5.47 eV。500K環境下でも 4.25 eV の論理閉包を維持し、超高熱伝導率で散逸を加速する。
### 1.2 非ドーピング原則 (Non-Doping Policy)
* **定義:** 自由電子を供給する不純物ドープを「禁止」。4.25 eV 障壁内に中間準位（リークパス）を作らせないため。
* **代替要件:** ノード（演算点）は、電荷移動を伴わない「局所欠陥（NVセンター）」または「同位体格子歪み」によって形成する。

## 2. 構造仕様 (Structural Specifications)
* **2.1 非時間的マッピング・グリッド:** 従来のゲート積層を排し、ダイヤモンド格子内に固定されたノード間結合重み（アテンション）を物理的な歪み干渉として実装。
* **2.2 熱的エンジン層:** 300K - 500K の熱場を供給するグラフェン/AlN製ヒートレール。熱的ゆらぎを局所解脱出の動力として活用。
* **2.3 分散読み出し共振器:** 各ノードに近接配置された、エネルギー交換を伴わない「分散結合型」の共振器。

## 3. 動作仕様 (Operational Specifications)
* **3.1 注入 (Injection):** 10ns以下の超短パルスにより、系を J=Max のカオス状態へ強制励起。
* **3.2 自律緩和 (Relaxation):** 500Kの熱エンジン下で、系が自律的にアトラクタへ沈殿。外部制御を排し物理法則に答えを委ねる。
* **3.3 掠奪 (Plunder):** Fisher情報量消失検知時、分散読み出しプロトコルを実行。ノードの固有振動数から離調したプローブ光を用い、位相シフトのみを非破壊でラッチする。

## 4. 運用・拘束条件 (Constraints)
* **温度下限:** 300K未満は緩和停滞を招くため「非推奨」。
* **閉包性:** 4.25 eV 未満の材質使用は「因果の漏出」を招くため禁止。
* **観測:** 緩和途中の介入は波束を収縮させ、公理1を阻害するため排除。
