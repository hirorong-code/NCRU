# NCRU 要求仕様書 (Requirements Specification) v1.0
**プロジェクト:** PMNT-OQC (Phase Mapping Network Theory - Optimized Quantum Computation)
**ステータス:** 最終確定版

## 1. 開発目的
設計されたハミルトニアン $H_{target}$ において、物理系が安定状態（アトラクタ）へ遷移する「不可避な緩和」を非時間的に観測し、その最終位相分布を「解」として掠奪するためのハードウェア・ユニット **NCRU (Non-Computational Resource Unit)** を定義する。

## 2. 実現範囲 (Scope)

### 2.1 実現可能なこと (Capabilities)
- **同型写像 (Isomorphic Mapping):** 論理問題を位相空間上のポテンシャルの谷（アトラクタ）として物理実装。
- **自律的緩和 (Autonomous Relaxation):** 系のリャプノフ性 ($\dot{J} \leq 0$) のみによる状態遷移。
- **終焉検知 (Termination Sensing):** 情報の熱的死 ($I(\theta) = 0$) の物理的判定。
- **非時間的抽出 (Atemporal Extraction):** 中間過程を排した「初期注入」と「最終抽出」のみの運用。

### 2.2 実現不可能なこと (Constraints)
- **動的因果制御:** 緩和途中での論理ゲート割り込み（因果系列の書き換え）。
- **可逆演算:** エントロピー増大を利用するため、演算過程の逆追跡は不可能。

## 3. 主要目標値 (Target Metrics)

| 項目 | 目標値 | 備考 |
| :--- | :--- | :--- |
| **閉包エネルギー障壁** | **4.25 eV** | 局所閉包を維持し、外界の干渉を遮断する最小分解能。 |
| **動作温度帯** | **300K - 500K** | 緩和を加速させるための熱的動作環境。 |
| **緩和速度 ($\tau_{relax}$)** | **< 10 ns** | 注入からアトラクタ到達までの物理スパン。 |
| **情報消失閾値 ($I_{limit}$)** | **$10^{-6}$ bit** | 計算完了と見なす Fisher 情報量の残余上限。 |

## 補足資料：熱的動作環境における論理的整合性

### 1. 熱エネルギーの「燃料化」
300K - 500K の熱エネルギー $k_B T$ は「ノイズ」ではなく、Lyapunov 関数 $J$ の最小化を加速させる**「熱的キネティクス」**として定義する。熱的ゆらぎは、系を偽のアトラクタから脱出させ、真の定常点へ導く「自然なアニーリング」として機能する。

### 2. 4.25 eV による「熱的隔離」の証明
動作温度下の最大熱エネルギー（約 0.043 eV）に対し、閉包障壁 4.25 eV は **100倍以上の強度** を持つ。これにより、ハミルトニアンの構造は熱的に書き換えられることなく維持される。

### 3. 低温環境の拒絶
極低温環境（mK帯）は、系の緩和を停滞させ、略奪効率を低下させるため「非推奨」とする。
