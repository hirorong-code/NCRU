# PMNT-OQC 要件定義 追記事項（動的変調プロトコル）Rev.1.95-Addendum
**プロジェクト:** Phase Mapping Network Theory - Optimized Quantum Computation
**目的:** 「1問題1チップ」の制約を突破し、単一基板での汎用計算・変数制御を実現する。

## 1. 構造仕様への追記 (Structural Addendum)
### 2.4 動的変調ポート (Dynamic Modulation Ports)
物理的に固定されたダイヤモンド格子に対し、外部から論理的変数を投影するためのインターフェース。

* **2.4.1 光学的格子制御 (Optical Lattice Tuning):**
    * **構成:** 各ノード（NVセンター等）に対し、特定波長の制御光を導入する微細導波路を積層。
    * **機能:** 光誘起による局所的な電子状態変化を利用し、隣接ノード間の「実効的な結合強度」を 0.1ns 単位で変調する。
* **2.4.2 磁気的ゼーマン・グリッド (Magnetic Zeeman Grid):**
    * **構成:** チップ外周および基板下層にマイクロ磁気コイル・アレイを配置。
    * **機能:** 局所的な磁場勾配を形成。各スピンノードのエネルギー準位（ゼーマン分離）を個別にオフセット（偏向）させ、ハミルトニアンに任意の「バイアス項」を物理的に加算する。

## 2. 動作仕様への追記 (Operational Addendum)
### 3.1.2 外部バイアス・キャリブレーション (External Bias Calibration)
自律緩和を開始する直前の、論理空間の「投影」プロセス。

* **手順:**
    1. **問題写像の分解:** ターゲットとするハミルトニアン $H_{target}$ を、「固定格子歪み $H_{static}$」と「外部動的変調 $H_{dynamic}$」の和として演算ユニット外（ASIC側）で分解する。
    2. **同期注入:** 高エネルギーパルス注入（Injection）と同時に、2.4.1 および 2.4.2 のポートから磁場・光パターンを空間的に投影。
    3. **フィールド・ロック:** 閉包（Closure）が完了し緩和が開始される瞬間、外部変調パターンを「固定」し、系を純粋な自律沈殿に委ねる。

## 3. 工学的インパクト (Engineering Impact)
* **汎用性の獲得:** 物理構造を再製造することなく、係数の異なる同種の問題群（例：異なるパラメータを持つ最適化問題）を1チップで処理可能。
* **開発サイクルの短縮:** アルゴリズムの微調整を、物理的な再配置ではなく「電磁気的なパラメータ更新」によって実現。
* **探索精度の向上:** 緩和の初期段階でバイアスを動的に変動させる（物理的なシミュレーテッド・アニーリング）ことで、より深いグローバルな解（谷）への誘導が可能。
