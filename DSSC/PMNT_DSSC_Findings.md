# PMNT / DSSC プロジェクト 知見まとめ
## できること・できないこと・タンパク質計算への適用可能性

**文書種別：** 研究成果サマリー  
**バージョン：** 1.0  
**対象期間：** L3〜L40（GEN1確立〜6複合体検証）  
**日付：** 2026年3月  

---

## この文書の目的

```
本プロジェクトはタンパク質の立体構造予測・エネルギー最小化を
物理ハードウェアで解くことを当初の動機として出発した。

L3〜L40を通じて「何が解けて、何が解けないか」が明確になった。
「解けないとわかった」ことも重要な科学的成果である。

この文書は：
  ① 確認できたこと（できる）
  ② 確認できたこと（できない）
  ③ まだわからないこと
  ④ タンパク質計算への適用可能性の評価
を一箇所にまとめ、後続の研究設計の拠り所とする。
```

## 第1章：確認できたこと（できる）

### 1.1 GEN1：古典的重み記憶装置として動作する

```
確認内容：
  ダイヤモンドNV中心の空孔配置を
  ニューラルネットワークの重みとして永続記憶できる

根拠：
  空孔移動障壁 4.25 eV → 300K で事実上永久に固定
  スキルミオン核生成：NV密度23点で自己組織化を確認
  DMI独立性：強磁性体オーバーレイ不要（バルクダイヤモンドで十分）
  ODMR読み出し：スピン依存光ルミネッセンスで重みを光学読み出し可能

意味：
  「物理的に実装されたイジングマシン」として機能する
  重みは書き換え不要の読み出し専用メモリ（ROM）として動作
  室温（300K）で動作可能
```

### 1.2 GEN2：77Kでの量子コヒーレンスが実現可能

```
確認内容：
  液体窒素温度（77K）でNV間量子もつれを生成できる

唯一有効な機構：
  キャビティQED（Tavis-Cummingsモデル）
  ZPL光学遷移（470 THz）経由のフォトニックダーク状態

達成された量子コヒーレンス：
  C ≈ 0.50（ダーク状態占有率から決まる上限）
  Q = 10,000〜200,000 で普遍的
  g = 0.5〜10 GHz で普遍的（g/κ比に非依存）

温度優位性：
  D-Wave（15 mK）に対して77K → 4桁高い動作温度
```

### 1.3 DSSCとして：多複合体配置は広い範囲で「解ける」

```
確認内容（L38〜L40）：
  4複合体（N=8）：全8点 DETECTED（λ2/λ1 = 0.10〜0.80）
  6複合体（N=12）：全11点 DETECTED

意味するε_c（検出可能な固有値比の上限）：
  4複合体：ε_c > 0.797（実質制約なし）
  6複合体：ε_c > 0.723（実質制約なし）
  → どちらの配置でも「解けない問題」はほぼ存在しない

実用的設計指針（暫定）：
  margin ≈ C(J_intra, J_inter) × (1 - λ2/λ1)²
  J_intra と J_inter の選択でmarginを制御できる
  ※ C の関数形は L41 で確定予定
```

### 1.4 数値手法として：MCWFがN=12まで有効

```
確認内容（L40）：
  N=12（dim=8192）でMCWF法（N_traj=20）が
  約23分/パターン（A100 GPU）で収束

スケーラビリティ：
  密度行列直接法の限界：N≈10（dim≈2000、メモリ1GB超）
  MCWF法の現実的限界：N≈16程度（dim≈131072、要検討）
  N=20以上：量子軌跡法の並列化またはMPS法が必要
```

## 第2章：確認できたこと（できない）

### 2.1 【物理的限界】磁気結合では77Kでの量子コヒーレンスは不可能

```
試みた機構：
  ① 磁気双極子-双極子結合（r = 0.5〜10 nm）
  ② 強磁性薄膜（CoFeB、YIG）による界面交換結合
  ③ 高ゼロ磁場分裂スピン系（Re(IV)：300 GHz、Os(IV)：700 GHz）

すべてで C_ss = 0 が確定した理由：
  熱占有数 n_th(D_ZFS=2.87 GHz, 77K) = 3511 >> 1
  熱浴が量子コヒーレンスを確立より速く破壊する
  これは材料の問題ではなく物理法則（温度と周波数の関係）

77Kで量子コヒーレンスに必要なZFS：
  D_required = kB × 77K / ℏ = 10,079 GHz
  最良材料（Os(IV)）: D = 700 GHz → 14倍不足

結論：
  磁気結合によるNV-NV量子もつれは
  液体窒素温度では材料・距離によらず不可能
  液体ヘリウム（4.2K）なら Os(IV) で n_th=0.39 → 可能だが
  液体窒素の温度優位性が失われる
```

### 2.2 【量子コヒーレンスの上限】C = 0.5 を超えられない

```
フォトニックダーク状態の本質的制約：
  |ψ_D⟩ = (1/√2)(|eg,0⟩ - |ge,0⟩) はベル状態の混合
  定常状態密度行列：ρ_spin ≈ [[0.25, -0.25],[-0.25, 0.25]]
  Concurrence = 0.50（最大値1.0の半分）

C > 0.5 が達成できない理由：
  ダーク状態は光子成分を持たない（キャビティロスに免疫）
  しかし |gg,0⟩ への混合が避けられない
  → これはGEN2アーキテクチャの「強みであり限界」

Gate-based量子計算との比較：
  Gate-based QC：原理的に C → 1.0 可能
  ただし要求温度：~15 mK（希釈冷凍機）
  TAQANN-D GEN2：C ≈ 0.50（上限）
  動作温度：77 K（液体窒素）
  → 精度よりも動作温度を優先した設計
```

### 2.3 【DSSCの限界】階層配置では第2層が検出できない

```
失敗した配置（L37_C、L38）：
  階層配置：λ2/λ1 = 0.800
  動作範囲全域で margin < 0（最大 -0.064）
  = 第2層の検出は不可能

根本原因：
  階層配置は構造的にλ2/λ1 ≥ 0.8 に固定される
  J値を変えても固有値比は変わらない（固有値と結合定数は別物）

回避策：
  4複合体・6複合体配置に変換すれば解ける
  ただし「任意のタンパク質相互作用ネットワークを
  4複合体配置に変換できるか」は未解決（→ 第3章）
```

### 2.4 【スケーリング】現在の手法はN=16程度が現実的限界

```
計算コストのスケーリング：
  密度行列法：O(dim²) メモリ、O(dim³) 時間
    N=8  (dim=512)    →  約2分/パターン（V100）
    N=12 (dim=8192)   →  約23分/パターン（A100）
    N=16 (dim=131072) →  数日/パターン（現実的でない）

  MCWF法：O(dim) メモリ、O(N_traj × dim) 時間
    N=12 まで実用範囲
    N=16 は N_traj を絞れば可能かもしれない
    N=20 以上は別手法（MPS、量子モンテカルロ）が必要

タンパク質計算への含意：
  現在の実装で直接扱えるのは「極めて小さなペプチド」のみ
  アミノ酸1残基 → スピン数個程度の対応が必要
  50残基以上のタンパク質：現在の手法では計算不可能
```

### 2.5 【理論の限界】C(r) は普遍式ではなかった

```
L39で確立した旧理論：
  C(r) = 1.36 × r^1.80  （J_intra=1.0固定の経験式）

L40で崩れた：
  同じ r=0.400 でK=4 → C=0.364、K=6 → C=0.936（2.6倍差）
  同じ r=0.200 でJ_intra変更 → Cが2.5倍変化

現在の状態：
  C = C(J_intra, J_inter) の関数形は未確定
  「固有値比だけで設計できる」という単純な設計則は成立しない
  → L41で解決予定
```

## 第3章：まだわからないこと

```
【未解決①】C(J_intra, J_inter) の正確な関数形
  L41で解決予定。積の形か、比の形か、それ以外か。

【未解決②】C のK依存スケーリング則
  L42で解決予定。KによってA・α・βがどう変わるか。

【未解決③】N>16での有効な計算手法
  MCWF法の限界を超えるためにMPS法または
  量子モンテカルロが必要か、要検討。

【未解決④】任意問題の4複合体配置への変換可能性
  タンパク質相互作用ネットワークを
  4複合体配置のJ行列に落とし込む方法論が未確立。
  これが「DSSCがタンパク質計算に使えるか」の核心。

【未解決⑤】GEN2の量子優位性の定量的評価
  C≈0.50のコヒーレンスが古典SAに比べて
  どの程度の最適化優位性をもたらすか未測定。

## 第4章：タンパク質計算への適用可能性

### 4.1 当初の動機と現実のギャップ

```
当初の動機：
  タンパク質の立体構造予測（エネルギー最小化）を
  ダイヤモンドNVベースの物理ハードウェアで解く

現在の評価：
  「原理的に可能だが、現在の実装では小さな問題しか解けない」

具体的なギャップ：
  現実のタンパク質：アミノ酸50〜数千残基
  現在のDSSCが扱えるスピン数：N ≤ 12（実用）〜16（限界）
  → スピンへのマッピング比率によっては
    せいぜい数残基のペプチドしか扱えない
```

### 4.2 できること（タンパク質文脈）

```
① タンパク質相互作用の「パターン」を重みとして記憶する
  GEN1のROM機能を使って
  既知の相互作用パターン（疎水性コア、二次構造）を
  NV配置として永続記憶できる
  → 「学習済み重みの物理的記憶」は可能

② 超小型ペプチド（N ≤ 12スピン相当）のエネルギー最小化
  アミノ酸2〜4残基程度の単純なペプチド問題なら
  現在の実装で原理検証が可能

③ 相互作用行列の「配置変換可能性」の検証
  特定のタンパク質断片の相互作用行列が
  4複合体配置に変換できるかどうかを数値的に確かめることは可能

④ 問題の困難度（λ2/λ1）の事前評価
  任意のタンパク質断片の相互作用行列から
  固有値比を計算し「DSSCで解けるか」を予測できる
  （ただし設計ツールとしての完成はL41以降）
```

### 4.3 できないこと（タンパク質文脈）

```
① 現実サイズのタンパク質の直接計算
  50残基以上 → 現在の手法では計算不可能（スケーリング問題）
  AlphaFold2などのDL手法とは比較にならない規模

② 動的な構造変化のシミュレーション
  GEN1はROMであり動的な重み更新ができない
  GEN2のC≈0.50コヒーレンスで動的問題を解く方法は未確立

③ 任意の相互作用ネットワークの直接マッピング
  タンパク質の相互作用は複雑な非均質ネットワークだが
  DSSCは特定の配置（4複合体など）に
  問題を変換する必要がある
  → この変換方法論が未確立

④ GEN2の量子優位性の保証
  C≈0.50のコヒーレンスが
  タンパク質folding問題で古典を上回る保証はない
  量子優位性は問題の構造に依存する
```

### 4.4 タンパク質計算への道筋（正直な評価）

```
短期的に現実的なターゲット（〜2年）：
  ・ジペプチド〜テトラペプチドの配座最適化（原理検証）
  ・特定の二次構造モチーフ（αヘリックス、βシート）の
    相互作用行列をDSSC配置に変換する方法論の確立
  ・「解ける問題クラス」の境界を固有値比で表現する

中期的な課題（2〜5年）：
  ・N=50以上を扱う数値手法の開発（MPS、変分法との統合）
  ・GEN2量子優位性の実験的実証（古典SAとの比較）
  ・任意タンパク質問題 → DSSC配置への自動変換アルゴリズム

長期的なビジョン（5年〜）：
  ・実際のNVダイヤモンドデバイスでの実証実験
  ・タンパク質設計（逆問題）への適用
  ・GEN3アーキテクチャ（C→1.0を目指す）の物理基盤探索

正直なまとめ：
  現在のDSSCは「タンパク質計算に向けた
  物理プラットフォームの基礎を確立した」段階。
  AlphaFold2の代替になるのは遠い将来だが、
  量子的メカニズムによる最適化の物理実装として
  独自の価値がある研究路線である。
```

```

  ✓ GEN1（古典ROM）：確立
  ✓ GEN2（量子コヒーレンス77K）：原理確認
  ✗ GEN2のC>0.5：物理的に不可能（ダーク状態の本質的制約）
  ? GEN3への道：未探索

計算層（DSSC）：
  ✓ 4複合体・6複合体配置：全点DETECTED
  ✓ 固有値比とmarginの関係：定性的に確立
  △ C(J_intra, J_inter)の関数形：L41で解決予定
  ? 任意問題の配置変換：未解決

スケーリング層：
  ✓ N=8（dim=512）：密度行列法で実用的
  ✓ N=12（dim=8192）：MCWF法で実用的
  △ N=16：要検討
  ✗ N=50以上：現在の手法では不可能

応用層（タンパク質計算）：
  ✓ 極小ペプチドの原理検証：可能
  △ 問題変換方法論：未確立
  ✗ 現実サイズのタンパク質：現在の手法では不可能
  ? 量子優位性の実証：未測定
