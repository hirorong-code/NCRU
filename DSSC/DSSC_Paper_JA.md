### 1.1 ノイマン型の限界

しかしノイマン型には本質的な制約がある。問題サイズNに対して計算量がO(N^k)以上で増加し、特に組み合わせ最適化やクラスター検出は指数的に困難になる。

### 1.2 DSSCの計算原理

```
ノイマン型：
  人間が「何をするか」を書く

DSSC：

  量子臨界点s*で答えを読み出す
```

DSSCでは「計算とは物理の自然な帰結である」。エネルギー最小化という普遍的な物理原理が計算を実行する。

### 1.3 既存技術との位置づけ

## 3. 計算クラスの定義

### 3.1 DSSCが解ける問題

```
入力：J_{ij} ∈ R^{N×N}（対称行列）
処理：エネルギー最小化（物理）
出力：C_{ij}（相関行列）
完了条件：s = s*

解ける条件：
  ① クラスター間でJ_{ij}が分離
    （境界をまたぐ縮退がない）
  ② 動作範囲：J_eff×J_{ij}×t* < π/4
  ③ 符号反転不要
```

### 3.2 古典アルゴリズムとの対応

```
スペクトルクラスタリング（古典）：
  J_{ij}のラプラシアンを構成
  固有ベクトルを計算 → O(N³)
  k-meansで分類

DSSC：
  J_{ij}をH_effに設定
  物理がエネルギー最小化
  s*でC_{ij}を読み出す
  = 「スペクトルクラスタリングの
     物理的アナログ」
```

### 3.3 DSSCが解けない問題

```
① 縮退問題：
   J_AC = J_BC のとき C_AC = C_BC
   境界をまたぐ縮退は解除不可

② 符号問題：
   C_{ij} > 0 のみ（強磁性H_XY）
   MAX-CUT等は解けない

③ 動作範囲超過：
   J_{ij}が大きすぎると
   C_{ij} ∝ sin²が折り返す
```
