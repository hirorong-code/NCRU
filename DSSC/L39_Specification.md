## 第1章：2つの問いの物理的統一

### 1.1 係数Cの変動の原因仮説

```
margin = C × (1-λ2/λ1)^2

2次摂動論が成立する条件：
  「摂動H_Γが小さい」
  = エネルギー差(λ1-λ2)が十分大きい

λ2/λ1=0.500（P3）：
  λ1-λ2 = 0.5 × J_eff × λ1（大）
  → 2次摂動論が成立
  → C ≈ 0.36（理論値0.38に近い）✓

λ2/λ1=0.692（P1）：
  λ1-λ2 = 0.308 × J_eff × λ1（小）
  → 高次摂動項が効いてくる
  → C > 0.38（過小評価）
  = 実測C=1.033（2次摂動論の2.7倍）

λ2/λ1=0.273（P4）：
  λ2/λ1は小さく有利なはず
  でも実測C=0.157（最小）
  = 別の効果が働いている
  → sin²動作範囲の問題
```

### 1.2 sin²動作範囲との競合

```
DSSCの動作原理：
  C_{ij} ∝ sin²(J_eff × J_{ij} × t*)

J_inter=0.20（P4）のとき：
  J_eff × J_inter = 308 × 0.20 = 61.6 MHz
  複合体間のJ_{ij}が大きすぎる

  sin²の引数が大きくなりすぎると：
  → C_{ij}が非単調になる
  → margin が下がる

  = J_inter=0.20は
    sin²の線形領域を超えている可能性

J_inter=0.10（P3）のとき：
  J_eff × J_inter = 308 × 0.10 = 30.8 MHz
  → sin²の線形領域内
  → C ≈ 0.36（正常）

J_interの設計上限の仮説：
  J_eff × J_inter × t* < π/4 の条件から
  J_inter < π/(4 × J_eff × t*) ≈ 0.12〜0.15
```

### 1.3 統一的な理解

```
marginを決める2つの競合効果：

効果A（固有値比効果）：
  λ2/λ1が小さいほどmarginが大きい
  2次摂動論で説明：C ≈ 0.38 × f(λ2/λ1)

効果B（動作範囲効果）：
  J_interが大きいとsin²範囲を超えてmarginが下がる
  閾値：J_inter ≈ 0.12〜0.15

実測の解釈：
  P1(λ2/λ1=0.692)：効果Aで高い（C=1.03）
  P3(λ2/λ1=0.500)：効果A・B共にバランス（C=0.36）
  P4(λ2/λ1=0.273)：効果Bで下がる（C=0.16）

→ 最適なJ_interは
  「効果Aと効果Bのバランス点」に存在する
```

## 第2章：実験設計

### 2.1 実験の構造

```
2軸スキャン設計：

軸1：J_interを0.03〜0.30まで細かくスキャン
     = 固有値比とsin²範囲を同時に変化させる
     = 両効果の「合力」を測定

軸2：J_intraを変化させてJ_interを固定
     = λ2/λ1を変えずにsin²範囲だけ変化させる
     = 2つの効果を分離する
```

### 2.2 Phase1：J_interスキャン（軸1）

**目的：** 「効果Aと効果Bの合力」を全域で測定

| No | J_inter | λ2/λ1 | 理論margin | 役割 |
|----|---------|--------|-----------|------|
| 1 | 0.03 | 0.797 | 0.016 | 高λ2/λ1の挙動 |
| 2 | 0.05 | 0.692 | 0.036 | L38P1再確認 |
| 3 | 0.08 | 0.568 | 0.071 | L38P2再確認 |
| 4 | 0.10 | 0.500 | 0.095 | 基準ライン |
| 5 | 0.12 | 0.442 | 0.118 | 効果B出始め？ |
| 6 | 0.15 | 0.368 | 0.152 | 設計上限付近 |
| 7 | 0.20 | 0.273 | 0.201 | L38P4再確認 |
| 8 | 0.30 | 0.143 | 0.279 | 明らかに範囲外 |

**判定基準：**
```
marginが単調増加：
  → 効果Bはまだ効いていない
  → J_inter上限は0.30以上

marginがピークを持つ：
  → ピーク位置が最適J_inter
  → 効果Aと効果Bのバランス点

J_inter > 閾値でmarginが減少：
  → 閾値がJ_inter設計上限
```

### 2.3 Phase2：J_intraスキャン（軸2）

**目的：** 効果Aと効果Bを分離する

```
J_inter=0.10（固定）で
J_intraを変化させる：

J_intra変化 → λ2/λ1変化 → 効果A変化
J_intra変化 → J_eff×J_intra変化 → 効果B変化

でも効果Bの条件：
  J_eff × J_inter × t* < π/4
  = J_interに依存（J_intraに依存しない）
```

| No | J_intra | J_inter | λ2/λ1 | 効果A | 効果B |
|----|---------|---------|--------|-------|-------|
| 1 | 0.50 | 0.10 | 0.273 | 大 | 同じ |
| 2 | 1.00 | 0.10 | 0.500 | 中 | 同じ |
| 3 | 2.00 | 0.10 | 0.692 | 小 | 同じ |

```
効果Bが同じで効果Aだけが変わる
→ marginの差が効果Aだけの寄与
→ Cのλ2/λ1依存性が純粋に測定できる
```

```
N_SPIN = 8, N_FOCK = 2, dim = 512
読み出し点：s* = 0.890
収束条件：sigma < 1×10⁻³
実行環境：高火力DOK V100（32GB）
```

## 第3章：期待される結果と判定

### 3.1 係数Cの理論的予測

```
2次摂動論の限界：
  λ2/λ1 ≤ 0.5：C ≈ 0.38（一定）
  λ2/λ1 > 0.5：C > 0.38（高次項が効く）

高次摂動論の補正：
  C(r) ≈ C₀ × (1 + α×r + β×r² + ...)

Phase2の軸2データから：
  C₀、α、βを実験的に決定する
```

### 3.2 J_inter設計上限の予測

```
sin²動作範囲の条件：
  J_scale × J_inter × DT × steps < π/4

推定上限：
  J_inter < 0.12〜0.15

Phase1のmarginがピークを持つJ_interが
実験的なJ_inter上限
```

### 3.3 設計原則の更新条件

```
L39完了後に確定する設計原則：

margin(J_inter, λ2/λ1) = 
  C(λ2/λ1) × (1-λ2/λ1)^2 × f(J_inter)

  C(λ2/λ1)：固有値比依存の係数
  f(J_inter)：動作範囲の補正係数
              f≈1（J_inter<上限）
              f<1（J_inter>上限）

最適設計点：
  df/dJ_inter × C(λ2/λ1) × (1-λ2/λ1)^2 = 0
  = marginが最大になるJ_inter
```
