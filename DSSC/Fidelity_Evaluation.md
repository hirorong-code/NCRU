# 忠実度定量化 候補評価と分岐点整理
## エネルギー最小化問題を目的とした場合の3候補比較

**文書種別：** 研究方針決定文書（Decision Document）  
**バージョン：** 1.0  
**日付：** 2026年3月  
**位置づけ：** L42仕様書確定前の方針決定  
**目的の問題：** タンパク質のエネルギー最小化（最安定構造探索）

---

## 第1章：3候補の定義

### 候補X：行列近似誤差（Matrix Approximation Error）

```
定義：
  元のJ行列 J_original と
  変換後のJ行列 J_converted の差のノルム

  忠実度X = 1 - ||J_original - J_converted||_F / ||J_original||_F

  忠実度X = 1.0 : 変換前後のJ行列が完全一致
  忠実度X = 0.0 : 全く異なるJ行列

計算方法：
  J_converted は W_intra（ペア内平均）と
  W_inter（ペア間平均）で均一化した行列
  → 計算コスト：O(N²)、シミュレーション不要
```

### 候補Y：エネルギーランドスケープ保存度（Landscape Preservation）

```
定義：
  元のJ行列と変換後のJ行列で
  全スピン状態のエネルギー順位がどれだけ一致するか

  忠実度Y = スピアマン順位相関（-1 〜 +1）

  忠実度Y = +1.0 : エネルギー順位が完全一致
             = 「同じ問題を解いている」
  忠実度Y =  0.0 : 順位に相関なし
             = 「全く別の問題になっている」

計算方法：
  全2^N 状態のエネルギー E = -Σ J[i,j] σ_i σ_j を計算
  元と変換後の順位リストのスピアマン相関を取る
  → N=8 なら 256状態、計算コスト：O(2^N × N²)
  → N=20 以上では困難
```

### 候補Z：最適解一致率（Optimal Solution Match Rate）

```
定義：
  元の問題の最適解（最低エネルギー状態）と
  DSSCの出力が一致するか

  忠実度Z = 一致した試行数 / 総試行数

  忠実度Z = 1.0 : 毎回正解を返す
  忠実度Z = 0.0 : 一度も正解を返さない

計算方法：
  ① 元のJ行列の最低エネルギー状態を全列挙で求める（N=8なら可能）
  ② DSSCを複数回実行して出力を記録
  ③ 一致率を計算
  → シミュレーション実行が必須（計算コスト大）
  → N=8なら実現可能、N=20以上は困難
```

## 第2章：エネルギー最小化問題への適性評価

### 2.1 候補X（行列近似誤差）

```
エネルギー最小化への適性：△

強み：
  ・計算が最も簡単（シミュレーション不要）
  ・変換の数値的品質を即座に確認できる
  ・分割アルゴリズムの改善指標として使いやすい
  ・「最低限の品質保証」として有用

弱み：
  ・J行列が数値的に近くても
    最低エネルギー状態が変わることがある
    例：J[1,2] = 0.8 → 0.6 に変わっただけで
        エネルギー最小状態のスピン配置が変わりうる
  ・「問題として同等か」を答えない
  ・Xが高くてもYが低い場合がありうる
    （数値は近いが問題の本質が変わっている）

エネルギー最小化問題における位置づけ：
  必要条件の一つだが十分条件ではない
  Yを測るための前段チェックとして有用
  「XがNGならYもNG」は言えるが
  「XがOKならYもOK」は言えない
```

### 2.2 候補Y（エネルギーランドスケープ保存度）

```
エネルギー最小化への適性：◎

強み：
  ・エネルギー最小化問題の本質を直接捉えている
    「変換後の最低エネルギー状態
     = 元の最低エネルギー状態」
    であることを保証する最も直接的な指標
  ・「同じ問題を解いているか」に答える
  ・部分的な保存（上位k位の順位が保存されるか）も
    計測可能で、精度をグラデーションで評価できる
  ・N=8なら全列挙で厳密に計算可能

弱み：
  ・N≥20では計算困難（2^N の爆発）
  ・「順位が保存されても一致率が低い」ことがある
    （DSSCが上位状態ではなく別の局所最適に陥る場合）
  ・スピアマン相関が高くても
    上位1位（最低エネルギー状態）だけが
    保存されないケースに気づけない可能性

エネルギー最小化問題における位置づけ：
  最も本質的な忠実度指標
  「変換が正しい」を証明するのに必要十分に近い
  ただしDSSCの動作（局所最適問題）は別途評価が必要
  → YとZの両方が高いときに初めて「使える」と言える
```

### 2.3 候補Z（最適解一致率）

```
エネルギー最小化への適性：◎（最終目標として）

強み：
  ・「実用的に使えるか」に直接答える
  ・エンドツーエンドの評価として最も信頼性が高い
  ・論文・応用の場面で最も説得力がある指標

弱み：
  ・元の問題の「正解」を古典的に求める手段が必要
    N=8 : 全列挙256通り → 可能
    N=20: 全列挙100万通り → 重いが可能
    N=50: 現実的でない
  ・シミュレーション実行が必須（費用・時間がかかる）
  ・「なぜ一致しないか」の原因が分からない
    （XもYも測っていないと改善方針が立てられない）
  ・局所最適問題とY（変換忠実度）問題を区別できない

エネルギー最小化問題における位置づけ：
  最終目標だが、今の段階で単独で測っても
  失敗したときの原因が特定できない
  X・Yを先に確立してからZを測るのが正しい順序
```

## 第3章：3候補の関係と測定順序

### 3.1 論理的な依存関係

```
X（行列近似誤差）
  ↓ 必要条件
Y（エネルギーランドスケープ保存）
  ↓ 変換の忠実度が保証されたら
DSSC の動作（margin、局所最適問題）
  ↓ DSSCが正しく動けば
Z（最適解一致率）

X ⊂ Y ⊂ Z（強さの順）
「ZがOK」→「YがOK（ほぼ）」→「XがOK（ほぼ）」
逆は成立しない
```

### 3.2 失敗したときの原因特定

```
ZのみOKでXYが未測定の場合：
  「なぜ動くか」「なぜ動かないか」が不明
  改善の手がかりがない

X失敗：
  → 変換（分割・均一化）に問題がある
  → グラフ分割アルゴリズムの改善が必要（L42課題）

X成功・Y失敗：
  → 数値は近いが問題の本質が変わっている
  → 均一近似の限界（不均一J行列への対応がL43課題）

X成功・Y成功・Z失敗：
  → 変換は正しいがDSSCが局所最適に陥っている
  → DSSCのアニーリングスケジュール改善が必要

X成功・Y成功・Z成功：
  → エネルギー最小化問題へのパイプライン完成
```

### 3.3 測定コストの比較

| 候補 | 計算方法 | N=8コスト | N=20コスト | シミュレーション要否 |
|------|---------|-----------|-----------|------------------|
| X | フロベニウスノルム | 瞬時 | 瞬時 | 不要 |
| Y | 全列挙+スピアマン相関 | 秒 | 数分 | 不要 |
| Z | DSSC実行×複数回 | 数分 | 数時間 | 必要 |

## 第4章：分岐点の整理と方針決定

### 4.1 確定事項（L38〜L41）

```
✓ margin > 0 → DSSCはDETECTEDを返す
✓ margin の予測精度：誤差16.4%（L41）
✓ 設計ツールとしてのmargin最大化は有効
✗ 変換の忠実度（X・Y）：未測定
✗ 最適解一致率（Z）：未測定
✗ DSSCが局所最適に陥る可能性：未評価
```

### 4.2 L42〜L44のロードマップ（案）

```
L42（現在設計中）：
  グラフ分割アルゴリズム + X・Y測定
  → 「変換がエネルギーランドスケープを保存するか」確認
  → N=8の小問題で全列挙と比較

L43：
  DSSC vs 古典SA（シミュレーテッドアニーリング）
  Z（最適解一致率）の初回測定
  → 「DSSCはエネルギー最小化に使えるか」直接検証
  → 局所最適問題の評価

L44：
  タンパク質模擬J行列でのパイプライン実証
  X・Y・Z全測定
  → 「タンパク質計算への橋渡し」
```

### 4.3 分岐条件

```
L42でYが低い（スピアマン相関 < 0.8）場合：
  → 均一近似では不十分
  → L43より先にL42'（不均一J行列対応）が必要
  → タンパク質計算への道はより遠い

L42でYが高い（スピアマン相関 ≥ 0.8）場合：
  → 変換は十分に忠実
  → L43（DSSC vs 古典SA）へ進める
  → DSSCの優位性評価が次の焦点になる

L43でZが低い（一致率 < 50%）場合：
  → DSSCが局所最適に陥っている
  → アニーリングスケジュールの改善が必要
  → 量子的優位性の主張には慎重になるべき

L43でZが高い（一致率 ≥ 80%）場合：
  → エネルギー最小化問題への適性が実証
  → L44（タンパク質模擬）へ進める
  → 論文執筆の基盤が整う
```

## 第5章：正直な現状評価

```
DSSCはエネルギー最小化問題に「向いている可能性がある」：
  理由：
    ・イジングモデルのエネルギー最小化が物理的動作原理
    ・量子アニーリングとの類似性
    ・L38〜L41でmarginの制御が確立済み

DSSCがエネルギー最小化問題に「向いているとは言えない」段階：
  根拠がないこと：
    ・変換の忠実度（X・Y）が未測定
    ・最適解一致率（Z）が未測定
    ・古典SAとの比較がない
    ・局所最適問題の評価がない
    ・タンパク質問題への適用実績がない

現在の正直な立ち位置：
  「DSSCで解けるかどうかを事前に予測できる
   設計ツールが確立した」（L41完了）
  
  次の問い：
  「DSSCで解いた答えは正しい答えか？」（L42〜L43）
  
  この問いに答えるための測定計画が
  X（必要条件）→ Y（本質的忠実度）→ Z（最終実証）
  という順序
```
